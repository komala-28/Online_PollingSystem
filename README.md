# Online Polling System

## Project Description

The **Online Polling System** is a web-based Java application designed to conduct online voting in a simple, secure, and efficient way. It allows users to participate in polls by selecting their preferred option and view results instantly.

This project is developed using **Java Servlets, JSP, JDBC, and MySQL**, and deployed on **Apache Tomcat Server**.

---

## How It Works

* Users log in using their credentials.
* After login, available poll options are displayed.
* Each user is allowed to cast only one vote.
* Votes are stored securely in a MySQL database.
* Results are calculated and displayed in real time.

---

## Purpose of the Project

* To digitize the traditional voting system
* To reduce manual errors in vote counting
* To provide fast and transparent results
* To demonstrate Java web application development

---

## Key Modules

### 1. User Module

* Login / Authentication
* Cast Vote
* View Results

### 2. Admin Module

* Manage Candidates / Options
* View Voting Statistics
* Control Polling System

### 3. Database Module

* Stores user details
* Stores votes securely
* Maintains data integrity
